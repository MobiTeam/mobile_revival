  • ФГБОУ ВПО «Нижневартовский государственный университет»
    В Нижневартовском государственном университете реализуются следующие уровни образования:
    высшее образование – бакалавриат;
    высшее образование – специалитет;
    высшее образование – магистратура;
    подготовка кадров высшей квалификации по программам подготовки научно-педагогических кадров в аспирантуре.
    Студенты университета представят проекты:
    • 1.Проект «Роботизированный манипулятор». Создание программно-аппаратного комплекса для управления роботизированным манипулятором.
    • 2.Проект «Робот – разносчик объектов». Создание робота на колесном шасси с манипулятором. Робот может перемещать легкие предметы, управляется со смартфона, имеет лицо, передает изображение с камеры на телефон, что позволяет управлять им, если он будет вне поля зрения.
    • 3.Проект «Робот удалённого доступа». Создание независимой роботизированной платформы, а также удобного и понятного инструмента на основе web-технологий для управления ею с использованием любого устройства с web-браузером.
    • 4.Проект «Прототип программно-управляемого автомобиля». Создание программно-аппаратного комплекса по управлению прототипом автомобиля.
    • 5.Проект «Управление компьютером с помощью смартфона». Создание программного обеспечения для управления мышью и клавиатурой компьютера под управлением операционной системы Windows с помощью android-смартфона.
    • 6.Проект «Робот с ориентацией внутри помещения». Создание программного обеспечения для мобильного устройства, позволяющего определить собственное местоположение внутри помещения, используя различные сенсоры.
